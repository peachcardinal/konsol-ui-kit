# Third-Party and Brand Asset Notice

This project includes UI assets that may be subject to third-party licensing and trademark rules.

## Potentially restricted assets

- Custom or commercial fonts (for example `Graphik`, `CoFo Sans Mono`)
- Brand and bank logos/icons
- Service/payment marks (for example `Sber`, `T-Bank/Tinkoff`, `Visa`, `Mastercard`)
- Exported design assets originating from private/internal Figma files

## Before making this repository public

1. Confirm redistribution rights for all font files and brand assets.
2. Remove or replace assets without public redistribution permission.
3. Keep only assets with explicit open/public license or owned rights.
4. Add explicit attribution/license text where required.
